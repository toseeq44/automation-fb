================================================================================
KEY INSIGHTS
================================================================================

1. Root Logger Level Matters
   - Must be DEBUG to capture all logging.info() calls
   - Without this, logs are filtered before handler sees them

2. Qt Signals Are Thread-Safe
   - Can emit from any thread
   - Slots run on correct thread context
   - Perfect for worker thread logging

3. Finally Blocks Are Essential
   - Always run, even on errors
   - Guarantee cleanup happens
   - Must handle exceptions in cleanup itself

4. Exception Handling in Cleanup
   - Cleanup code must not raise exceptions
   - Should have try/except around removals
   - Should always complete despite errors

5. Signal Disconnection Needs Safety
   - Check if connected before disconnecting
   - Wrap in try/except
   - Handle errors gracefully
