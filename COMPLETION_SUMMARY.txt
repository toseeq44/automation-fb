========================================================================
THE 5-STEP WORKFLOW
========================================================================

STEP 1: Load Credentials
  File: step_1_load_credentials.py
  Task: Read login_data.txt and validate credentials
  Error: CredentialsError (if file missing or invalid)

STEP 2: Find Browser Shortcut
  File: step_2_find_shortcut.py
  Task: Search desktop for browser .lnk shortcut file
  Error: ShortcutError (if shortcut not found)
  Supports: Chrome, Firefox, Edge, IX, GoLogin, Incogniton, Orbita

STEP 3: Launch Browser & Maximize
  File: step_3_launch_browser.py
  Task: Open shortcut and maximize browser window
  Functions: open_shortcut(), maximize_window()
  Error: BrowserLaunchError (if can't find window)
  Feature: Retries, mouse movement feedback

STEP 4: Check Session Status
  File: step_4_check_session.py
  Task: Detect if user is logged into Facebook
  Returns: SessionStatus.LOGGED_IN / LOGGED_OUT / UNKNOWN
  Method: Image recognition (profile icon or login form)

STEP 5: Handle Login/Logout
  File: step_5_handle_login.py
  Task: Logout if needed, then login with credentials
  Functions: logout(), login()
  Features: Click detection, form filling, human-like typing
