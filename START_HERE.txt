CONTENTFLOW PRO - Comprehensive Video Automation Suite

FEATURES:
  ✓ Multi-platform link grabbing (YouTube, Instagram, TikTok, Facebook, Twitter)
  ✓ Advanced video downloading with quality selection
  ✓ Professional video editing (trim, crop, filters, transitions)
  ✓ Metadata removal from videos
  ✓ Automated uploads using IX Browser
  ✓ License system with hardware binding
  ✓ Modern PyQt5 GUI (OneSoul Flow design)

MODULES:
  • link_grabber - Extract video URLs

  • video_editor - Edit and process
  • metadata_remover - Remove metadata
  • auto_uploader - Automate uploads
  • api_manager - Configure APIs
  • license - License validation
  • 4 other support modules

DEPENDENCIES:
  • yt-dlp, PyQt5, requests
  • moviepy, pillow, numpy

  • cryptography, browser_cookie3
  • 15+ other libraries
